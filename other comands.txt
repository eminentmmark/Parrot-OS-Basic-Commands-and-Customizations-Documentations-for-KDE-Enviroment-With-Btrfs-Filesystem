// repairing broken Packages
	
	
	
	
	
	
	1. Install BitLocker Support
	
	Blair should install:
	
	sudo apt install dislocker -y
	
	Dislocker allows Linux to unlock BitLocker drives.
	
	2. Create Mount Folders
	sudo mkdir -p /mnt/bitlocker
	sudo mkdir -p /mnt/windows
	3. Unlock the BitLocker Drive
	Using the BitLocker Password
	
	Run:
	
	sudo dislocker -V /dev/sda1 -u -- /mnt/bitlocker
	
	Blair will be prompted for the BitLocker password.
	
	OR Using Recovery Key
	sudo dislocker -V /dev/sda1 -pYOUR-RECOVERY-KEY -- /mnt/bitlocker
	
	Example:
	
	123456-123456-123456-123456-123456-123456-123456-123456
	4. Mount the Unlocked Volume
	
	After successful unlock:
	
	sudo mount -o loop /mnt/bitlocker/dislocker-file /mnt/windows
	
	Now Blair can access files in:
	
	/mnt/windows
	5. Open the Mounted Drive
	
	Using Dolphin:
	
	dolphin /mnt/windows
	6. Unmount When Finished
	sudo umount /mnt/windows
	sudo umount /mnt/bitlocker
	
	
	
	
	the main command to find a specific file is:
	
	find
	1. Find a File by Name in a Directory
	
	Syntax:
	
	find /path/to/directory -name "filename"
	
	Example:
	
	find ~/Documents -name "report.pdf"
	
	This searches:
	
	inside Documents
	
	for exactly:
		
		report.pdf
		2. Case-Insensitive Search
		
		Useful if Blair is unsure about uppercase/lowercase.
		
		find ~/Documents -iname "report.pdf"
		
		-iname ignores case.
		
		3. Search Entire System
		sudo find / -name "filename"
		
		Example:
		
		sudo find / -name "libreoffice"
		4. Find Only Files
		find ~/Downloads -type f -name "*.pdf"
		5. Find Only Folders
		find ~/Projects -type d -name "Media"
		6. Search by Extension
		
		Example:
		find all MP4 videos.
		
		find ~/Videos -type f -name "*.mp4"
		7. Search and Suppress Permission Errors
		find / -name "filename" 2>/dev/null
		8. Faster Alternative — locate
		
		Install:
		
		sudo apt install plocate -y
		
		Update database:
		
		sudo updatedb
		
		Then search:
		
		locate filename
		
		Much faster than find.
		
		9. Search for Partial Names
		
		Example:
		
		find ~/Documents -iname "*proposal*"
		
		Finds:
		
		proposal.docx
		badilisha_proposal.pdf
		finalproposal.txt
		
		etc.

// backups with rsync


						# create backup location

						mkdir -p /mnt/backup/system-config		{or u can use		mkdir -p /media/blair/backup/system-config}
						
						
						# basic system config backup

						sudo rsync -aAXv /etc /home /root /mnt/backup/system-config


						# Backup only configuration files (lighter)

						sudo rsync -a /etc /home/blair/.config /mnt/backup/system-config

						
						# Exclude unnecessary files (recommended)
						
						sudo rsync -aAXv \
						--exclude={"/dev/*","/proc/*","/sys/*","/tmp/*","/run/*","/mnt/*","/media/*","/lost+found"} \
						/ /mnt/backup/system-config
						
						
						# Verify backup integrity
						
						ls -lah /mnt/backup/system-config
						
						
						# restore later (usually for live USB)
						
						sudo rsync -aAXv /mnt/backup/system-config/ mnt/
